# template_modules

## English

This folder contains the full feature structure for the template modules domain. It groups the controller, service, module registration, Swagger documentation, DTOs, entity definitions, interfaces, and repository layer in one place.

## Tiếng Việt

Thư mục này chứa toàn bộ cấu trúc tính năng cho domain template modules. Nó gom controller, service, module đăng ký, tài liệu Swagger, DTO, entity, interface và tầng repository vào cùng một nơi.
