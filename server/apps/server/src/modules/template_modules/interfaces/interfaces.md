# interfaces

## English

This folder contains shared TypeScript interfaces and types for template modules. Interfaces define contracts used across the controller, service, repository, and other related files.

## Tiếng Việt

Thư mục này chứa các interface và type TypeScript dùng chung cho template modules. Interface định nghĩa hợp đồng dữ liệu được sử dụng giữa controller, service, repository và các file liên quan khác.
