# entities

## English

This folder contains entity definitions for template modules. Entities describe the persisted data model and map the feature data to database tables or collections.

## Tiếng Việt

Thư mục này chứa các định nghĩa entity của template modules. Entity mô tả mô hình dữ liệu được lưu trữ và ánh xạ dữ liệu của tính năng với bảng hoặc collection trong database.
