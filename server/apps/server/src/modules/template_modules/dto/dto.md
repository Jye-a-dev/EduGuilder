# dto

## English

This folder contains Data Transfer Object files for template modules. DTOs define the expected shape of request data for create, update, query, filtering, sorting, and pagination operations.

## Tiếng Việt

Thư mục này chứa các file Data Transfer Object của template modules. DTO dùng để định nghĩa cấu trúc dữ liệu request cho các thao tác tạo mới, cập nhật, truy vấn, lọc, sắp xếp và phân trang.
